### This blog is based on hexo.
未來UI跟底層架構可能會被我大改，但他現在是一個沒甚麼改過的模板。

### 本地作業

#### 更新
```bash
hexo clean
hexo generate
hexo deploy
```
#### 測試
```
hexo s
```
#### 更新
```
//Create a new post
$ hexo new "My New Post"

//Run server
$ hexo server

//Generate static files
$ hexo generate
```
